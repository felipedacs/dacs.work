# dacs.work
* Deploy in this repo on gh-pages

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Deploy on github
```
chmod +x deploy.sh
./deploy.sh
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
